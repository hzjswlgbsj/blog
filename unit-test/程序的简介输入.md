## 定义

什么是直接输入，什么是简介输入呢？直接看代码

```javascript
import { userAge } from "./user";

// 直接输入
function add(a: number, b: number) {
  return a + b;
}

// 间接输入
function doubleUserAge(): number {
  return userAge() * 2;
}
```
